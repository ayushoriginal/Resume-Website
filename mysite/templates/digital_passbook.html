A handy digital pass-book application for android platform. It helps the user keep track of transactions made across multiple bank accounts. This project was done to demonstrate Android App Development skills.

This project was done as a part of 80-hr training session on Android Application Development at Globsyn Finishing School, Kolkata in June 2013.

Eclipse IDE was used for designing GUI as well as coding. Custom styling for various GUI elements.
