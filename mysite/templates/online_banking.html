It is an online Banking System using PHP, MySQL, HTML5, CSS and JavaScript. No back-end framework has been used. This was made to demonstrate web-designing and database-programming skills.

This project was done as a part of an 80-hr training session on Web Development at Globsyn Finishing School, Kolkata during July 2013.

Features
Tools Used
    • A few basic security Aspects - SQL Injection, Password Hash and stuff
    • Two different Login and Transaction Password for added security
    • All forms validated on Client Side using JS and on Server Side using PHP
    • Nice Looking WebPages and custom Web Fonts - All CSS3 animation
    • No libraries or framework. Everything from scratch
    • Ability to send mails to Admin / other account holders
    • Netbeans was used as an IDE for the project
    • Dreamweaver was used for HTML5 pages
    • XAMPP was used for hosting and testing the website
    • MySQL was used as a Database Management System
    • Photoshop was used to edit images
    • Git was used as a version control system
